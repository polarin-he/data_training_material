# POLARIN Data Lifecycle Management Training Material
## Repository Content
This folder outlines strategies for managing data throughout its entire lifecycle, from initial creation to long-term preservation and sharing. It includes materials on planning and implementing sustainable data stewardship practices.

### Key Topics
- **Planning: Creating and maintaining data management plans (DMPs) and data policies**
- **Practising: Strategies for sustainable data publication and preservation**

### Structure of repository
1. **Learning Modules**: Focused on each phase of the data lifecycle — from initial collection in extreme polar environments to long-term preservation and open data sharing.
2. **Case Studies**: Real-world examples from polar research projects, illustrating best practices in handling diverse data types.
3. **Additional Resources**: Links to relevant (polar) data management tools (template generators, compliance checkers, conversion tools), standards, and publications.

# Data concepts
## Repository Content
This folder provides an introduction to the data concepts used later in the POLARIN training material. It outlines the FAIR data guiding principles and the FAIR data standards and formats.

### Key Topics
- **FAIR data guiding principles**
    - **Findable (F)**
    - **Accessible (A)**
    - **Interoperable (I)**
    - **Reusable (R)**
- **FAIR data standards and formats (including but not limited to)**
    - **CF-NetCDF**
    - **Darwin Core Archive**

### Structure of repository
1. **Learning Modules**
2. **Additional Resources**

## Note on the use of AI in these Jupyter Notebooks
The contents Jupyter Notebooks in this repository have partially been made using the GPT-4o Language Model developed by OpenAI, but the output is proofread by a gruop of data management experts connected to the POLARIN project. Learn more about the GPT-4o Language Model [here](https://openai.com/index/hello-gpt-4o/).

## Note on FAIRness of these Jupyter Notebooks
The Jupyter Notebooks in this repository are designed to be FAIR themselves. In designing these Jupyter Notebooks, the rules outlined in the "[FAIR for Jupyter Notebooks: A Practical Guide](https://ardc.edu.au/resources/fair-for-jupyter-notebooks-a-practical-guide/)" were taken into account.

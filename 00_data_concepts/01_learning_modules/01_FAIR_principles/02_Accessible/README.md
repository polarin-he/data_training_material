# FAIR guiding principles - **A**ccessible (A)
The FAIR guiding principles are divided into 4 categories, each with its own set of principles. The second category is ***A**ccessible. This category consists of 4 main principles, which are outlined below.

## Key principles covered in this directory
- **A1. (Meta)data are retrievable by their identifier using a standardised communications protocol**
    - **A1.1 The protocol is open, free, and universally implementable**
    - **A1.2 The protocol allows for an authentication and authorisation procedure, where necessary**
- **A2. Metadata are accessible, even when the data are no longer available**

## Note on FAIRness of these Jupyter Notebooks
The Jupyter Notebooks in this repository are designed to be FAIR themselves. In designing these Jupyter Notebooks, the rules outlined in the "[FAIR for Jupyter Notebooks: A Practical Guide](https://ardc.edu.au/resources/fair-for-jupyter-notebooks-a-practical-guide/)" were taken into account.

# FAIR guiding principles - **R**eusable (R)
The FAIR guiding principles are divided into 4 categories, each with its own set of principles. The fourth category is **R**eusable. This category consists of 4 principles, which are outlined below.

## Key principles covered in this directory
- **R1. (Meta)data are richly described with a plurality of accurate and relevant attributes**
    - **R1.1. (Meta)data are released with a clear and accessible data usage license**
    - **R1.2. (Meta)data are associated with detailed provenance**
    - **R1.3. (Meta)data meet domain-relevant community standards**

## Note on FAIRness of these Jupyter Notebooks
The Jupyter Notebooks in this repository are designed to be FAIR themselves. In designing these Jupyter Notebooks, the rules outlined in the "[FAIR for Jupyter Notebooks: A Practical Guide](https://ardc.edu.au/resources/fair-for-jupyter-notebooks-a-practical-guide/)" were taken into account.

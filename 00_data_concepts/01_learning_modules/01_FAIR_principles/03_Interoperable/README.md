# FAIR guiding principles - **I**nteroperable (I)
The FAIR guiding principles are divided into 4 categories, each with its own set of principles. The third category is ***I**nteroperable. This category consists of 3 main principles, which are outlined below.

## Key principles covered in this directory
- **I1. (Meta)data use a formal, accessible, shared, and broadly applicable language for knowledge representation.**
- **I2. (Meta)data use vocabularies that follow FAIR principles**
- **I3. (Meta)data include qualified references to other (meta)data**

## Note on FAIRness of these Jupyter Notebooks
The Jupyter Notebooks in this repository are designed to be FAIR themselves. In designing these Jupyter Notebooks, the rules outlined in the "[FAIR for Jupyter Notebooks: A Practical Guide](https://ardc.edu.au/resources/fair-for-jupyter-notebooks-a-practical-guide/)" were taken into account.

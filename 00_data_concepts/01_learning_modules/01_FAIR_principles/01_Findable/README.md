# FAIR guiding principles - **F**indable (F)
The FAIR guiding principles are divided into 4 categories, each with its own set of principles. The first category is **F**indable. This category consists of 4 principles, which are outlined below.

## Key principles covered in this directory
- **F1. (Meta)data are assigned a globally unique and persistent identifier**
- **F2. Data are described with rich metadata**
- **F3. Metadata clearly and explicitly include the identifier of the data they describe**
- **F4. (Meta)data are registered or indexed in a searchable resource**
